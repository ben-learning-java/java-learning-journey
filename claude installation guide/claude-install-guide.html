1

Install Homebrew

Homebrew is a package manager for macOS. Open Terminal and run:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
üîó Visit Homebrew.sh
2

Install Node.js

Use Homebrew to install Node.js:

brew install node
3

Install Claude Code

Install Claude Code globally:

npm install -g @anthropic-ai/claude-code

Start Claude Code:

claude

You'll need an API key from Anthropic Console

1

Install Chocolatey

Chocolatey is a package manager for Windows. Open PowerShell as Administrator and run:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
üîó Visit Chocolatey.org
2

Install Node.js

Use Chocolatey to install Node.js:

choco install nodejs
3

Install Claude Code

Install Claude Code globally:

npm install -g @anthropic-ai/claude-code

Start Claude Code:

claude

You'll need an API key from Anthropic Console
